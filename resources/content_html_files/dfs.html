Depth-First Search (DFS)

Depth-First Search (DFS) is a graph traversal algorithm that explores each branch of the graph as deeply as possible before moving to the next branch. In DFS, starting from a source node, we visit an adjacent node and recursively visit all its adjacent nodes until all reachable vertices from that node have been visited. Then we backtrack to visit any remaining unvisited adjacent nodes from the source.

DFS from a Given Source

The DFS algorithm can start from any given source node and explores all reachable vertices by going deeper into each branch. This approach is similar to the Depth-First Traversal of a tree. However, unlike trees, graphs can contain cycles, so DFS uses a boolean visited array to prevent revisiting nodes.

Steps of the DFS Algorithm

  • Start from the source node and mark it as visited.
  • For each adjacent unvisited node:
    • Recursively apply DFS on the node.
    • Continue until all reachable nodes from the source are visited.
  • Backtrack and move to any remaining unvisited adjacent nodes from the initial source node.

Complexity Analysis of DFS

  • Time Complexity: O(V + E), where V is the number of vertices and E is the number of edges in the graph.
  • Auxiliary Space: O(V + E), due to the visited array of size V and the stack space for recursive calls.
